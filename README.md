# Love Date Site

一个为情侣准备的约会邀请与共同回忆网站。每个人都可以在首页创建自己的专属空间，再把邀请链接和空间密码分享给 TA。

## 功能

- 约会邀请与可切换月份的日期选择
- 餐厅心愿单与约会准备清单
- 共同回忆时间线、照片和悄悄话
- 周年纪念日提醒与三套主题
- 专属空间：首次访问即可创建自己的空间，不需要公共演示模板
- 邀请访问：凭空间代码（或邀请链接）和空间密码进入对方的空间
- 云端同步：同一个空间的清单、回忆和设置会跨设备同步

## 本地使用

直接打开 `index.html` 即可运行。首次访问时，填写双方名字并设置空间密码，即可创建一个新的专属空间。

进入对方空间时，需要对方分享的邀请链接（或空间代码）和密码。密码不会显示在页面中，请通过可信方式单独告诉对方。

> 本项目已接入 Supabase 作为空间数据的同步服务。公开部署前，请为自己的 Supabase 项目配置相应的表、RPC 函数和安全策略。
