# Love Date Site

一个为情侣准备的约会邀请与共同回忆网站。你可以把它当作模板，替换名字、文案、日期和照片，做成属于两个人的小角落。

## 功能

- 约会邀请与可切换月份的日期选择
- 餐厅心愿单与约会准备清单
- 共同回忆时间线、照片和悄悄话
- 周年纪念日提醒与三套主题
- 本地保存：刷新页面后，当前浏览器中的内容仍会保留

## 本地使用

直接打开 `index.html` 即可运行。

初始演示密码为 `2026`，可在“我们”页面修改。

> 当前版本的数据保存在浏览器本地，适合演示和个人使用。若需要两个人跨设备实时同步，可在后续接入 Supabase 等后端服务。
